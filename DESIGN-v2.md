## 2. 타겟 유저 페르소나

### 페르소나 A: 직장인 이모 (35, 마케팅)
- "AI가 뭔지는 아는데 뭘 써야 할지 모르겠어"
- 글쓰기, 번역, 이미지 생성이 주 목적
- 예산: 무료~월 1만원 선호
- 클릭: AI 추천 퀴즈 → 가격 비교 → 프롬프트 저장소

### 페르소나 B: 대학생 박모 (22, 경영학)
- "과제에 AI 쓰고 싶은데 어느 게 좋아?"
- 요약, 분석, 프레젠테이션 자료 생성
- 예산: 무료 선호
- 클릭: 인기 랭킹 → 한국어 성능 비교 → 용어사전

### 페르소나 C: 프리랜서 최모 (28, 디자인)
- "이미지/비디오 AI 비교하고 싶어"
- 이미지 생성, 비디오 편집, 디자인 보조
- 예산: 유료도 OK (효율적이면)
- 클릭: 이미지 AI 비교 → 비디오 AI → 가격 비교

### 페르소나 D: 기술 관심인 김모 (40, 자영업)
- "컴퓨터에서 돌아가는 AI도 있는다고 하던데?"
- 로컬 AI 호기심, 개인정보 보호 중요
- 클릭: 내 PC 추천 → 로컬 AI 가능 여부 → 무료 모델 추천

## 4. 핵심 기능 정의

### 4.1 AI 추천 위젯 (NEW - 핵심 기능)

**위치**: 홈페이지 최상단 (Hero 영역)

**흐름**: 시작 -> Q1(용도) -> Q2(예산) -> Q3(환경) -> Q4(개인정보) -> 결과

**Q1: 용도 선택 (복수 선택 가능)**
- 글쓰기/번역, 코딩/개발, 이미지 생성, 비디오 생성, 문서 요약/분석, 대화/상담, 전부 다

**Q2: 예산**
- 무료만, 월 1만원 이하, 월 5만원 이하, 상관없음

**Q3: 사용 환경**
- 웹 브라우저만 (클라우드 API), 내 컴퓨터에서 실행하고 싶어 (로컬), 둘 다 궁금해

**Q4: 개인정보 중요도**
- 매우 중요, 보통, 상관없음

**결과**: TOP 3 추천 카드 (점수 기반: 용도 40점 + 가격 30점 + 한국어 15점 + 개인정보 15점)

### 4.2 내 PC 추천 (NEW)

**자동 사양 감지** (JavaScript):
- GPU 모델명: WebGL renderingInfo
- VRAM: GPU 모델명 DB 매칭
- CPU 코어수: navigator.hardwareConcurrency
- RAM: navigator.deviceMemory (Chrome)
- OS: navigator.platform

**VRAM DB**: RTX 4090(24GB), RTX 4080(16GB), RTX 4070(12GB) 등 30+ GPU

**로컬 AI 호환성**: VRAM 기준으로 7B/13B/32B/70B 모델 실행 가능 여부 판단

**추천 결과**: 로컬 가능 모델 + 무리하면 가능 + 클라우드 추천

### 4.3 기존 기능 유지/개선
- 가격 비교: 모델 카드에 "이 용도에 추천" 배지
- 벤치마크: 초보자용 해설 추가
- 뉴스/프롬프트/OpenClaw: 유지
